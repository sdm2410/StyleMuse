Aplicación web full stack para explorar y gestionar tendencias de moda. Permite visualizar, crear y administrar contenido mediante arquitectura cliente-servidor.

Funcionalidades
- Sistema de usuarios con roles y permisos
- CRUD de contenido (crear, ver, editar, eliminar)
- API REST para comunicación frontend-backend
- Renderizado dinámico en la interfaz
